# 🚀 Predictor V2.0

A futuristic **AI-style prediction app** built with a cyber / hacking-themed UI.
Designed for mobile-first experience with smooth performance and clean visuals.

---

## 📱 Overview

Predictor V2.0 is a mobile-focused application that simulates AI-based predictions for games like:

* 91 Club
* Diu Win
* Big Daddy
* Tiranga Game
* Goa Games
* OK Win
* BDG Game
* TC Lottery
* Jay Club

⚡ The app provides **randomized results** in a premium hacking-style interface.

---

## 🎨 Features

* 🔐 Subscription-based login system (UID)
* ⚡ Fast and lightweight (mobile optimized)
* 🎯 Big/Small prediction mode
* 🔢 Number prediction mode (1–9)
* 🧠 AI-style loading animation
* 🟢 Live status indicator (AI Signal - Online)
* 📲 Telegram integration for subscription
* 💾 Auto login (saved locally)
* 🧾 Scrollable disclaimer + instructions
* 🎨 Cyber / hacking UI theme (green-blue neon)

---

## 🧠 How It Works

1. Enter your **Subscription Key (UID)**
2. Select a game
3. Enter the **Period Number**
4. Click **RESULT**
5. Get a randomly generated output

## ⚙️ Setup Instructions

### Flutter App

1. Clone the repo
2. Add assets:

```
assets/images/
```

3. Update `pubspec.yaml`:

```yaml
flutter:
  assets:
    - assets/images/
```

4. Run:

```bash
flutter pub get
flutter run
```

---

### Web Version

1. Open `index.html`
2. Keep images in same folder or `/assets`
3. Run in browser or WebView
